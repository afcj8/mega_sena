<h1>🌐 Mega-Sena</h1>

<h2>⚡ Projeto</h2>
O programa realiza o sorteio da Mega-Sena, onde são sorteados 6 números, aleatoriamente e sem repetição, de um total de 60.

Link do programa: https://afcj8.github.io/mega_sena/
